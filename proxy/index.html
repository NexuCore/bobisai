AI Webhook Endpoint

This page acts as a webhook that processes incoming messages through the DeepSeek R1 model.

Endpoint Information

URL: This page's URL

Method: POST

Content-Type: application/json

Request Body: {"message": "Your message here"}

How to Use

Send a POST request to this page with a JSON body containing your message. The response will contain the AI's reply.

Example using curl:

curl -X POST \
  -H "Content-Type: application/json" \
  -d '{"message":"Hello"}' \
  [YOUR_PAGE_URL_HERE]
